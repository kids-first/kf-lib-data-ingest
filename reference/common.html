FileRetriever is a class for downloading the contents of remote files using whatever mechanism is appropriate for the given protocol.

class kf_lib_data_ingest.common.file_retriever.FileRetriever(storage_dir=None, cleanup_at_exit=True)[source]¶

A self-cleaning file contents downloader. Downloads contents of remote files to local temp files. When the FileRetriever instance loses scope, the temp files optionally go away.
